# Cloud Infrastructure Assignment

## Overview
This project is a simple internal project tracking application designed for deployment in a private cloud environment using Docker and Docker Compose.

## Components
- Frontend: HTML, CSS, JavaScript served by Nginx
- Backend: Node.js + Express API
- Database: MongoDB

## Deployment
The application is containerised and orchestrated with Docker Compose.

## Services
- frontend
- backend
- mongo

## Ports
- Frontend: 8080
- Backend: 5000
- MongoDB: 27017

## Run
Run the application using:
docker compose up --build

## Access
Open in your browser:
http://localhost:8080
